# User Personas

## Persona 1: Customer (매장 고객)

| 항목 | 내용 |
|---|---|
| **이름** | 매장 고객 (Customer) |
| **역할** | 테이블에 앉아 태블릿으로 음식을 주문하는 고객 |
| **목표** | 빠르고 편리하게 메뉴를 탐색하고 주문하기 |
| **동기** | 직원 호출 없이 원하는 시점에 즉시 주문, 추가 주문도 자유롭게 |
| **기술 수준** | 다양 (스마트폰/태블릿 기본 사용 가능) |
| **주요 태스크** | 메뉴 조회, 장바구니 담기, 주문 확정, 주문 내역 확인 |
| **불편사항** | 직원 대기 시간, 주문 누락, 복잡한 UI, 주문 상태 불투명 |
| **성공 기준** | 3번 이내 터치로 원하는 메뉴 찾기, 즉시 주문 확정 가능 |

## Persona 2: Admin (매장 관리자)

| 항목 | 내용 |
|---|---|
| **이름** | 매장 관리자 (Admin) |
| **역할** | 매장 운영을 총괄하며 주문과 메뉴를 관리하는 운영자 |
| **목표** | 실시간으로 주문을 모니터링하고 매장을 효율적으로 운영 |
| **동기** | 인건비 절감, 주문 처리 효율화, 매장 운영 가시성 확보 |
| **기술 수준** | 중급 (웹 기반 관리 도구 사용 경험) |
| **주요 태스크** | 주문 모니터링, 주문 상태 변경, 테이블 세션 관리, 메뉴 CRUD |
| **불편사항** | 주문 놓침, 테이블 상태 파악 어려움, 메뉴 변경 번거로움 |
| **성공 기준** | 신규 주문 2초 이내 확인, 테이블 상태 한눈에 파악, 메뉴 즉시 수정 |
